# Iran Victims Memorial

A living memorial to the victims of the 2025-2026 Iranian protests.

This project aims to document the names, ages, cities, and stories of those who lost their lives during the nationwide uprising.

## Structure
- `index.html` — Main page
- `css/styles.css` — Styles and animations
- `js/main.js` — Interactivity (filtering, modals, scroll effects)
- `data/victims.json` — Victim records (scalable)

## Sources
Data compiled from verified reports by:
- BBC News Persian
- Amnesty International
- Iran Human Rights Documentation Center (IHRDC)
- National Council of Resistance of Iran (NCRI/PMOI)

## Notes
This is an early version with 20 documented martyrs. We will gradually add more as verifiable information becomes available.
Each entry includes name, age, city, date of martyrdom (when confirmed), and a brief account.
Photographs are being collected with respect for families; placeholders will be replaced as we obtain consent.
